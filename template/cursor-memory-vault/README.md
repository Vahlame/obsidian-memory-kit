# Cursor Memory Vault

Vault base para memoria persistente en Cursor.

## Archivos principales

- `MEMORY.md`: conocimiento global durable.
- `SESSION_LOG.md`: bitacora cronologica.
- `PROJECTS/<proyecto>.md`: memoria por proyecto.
- `cursor-install/`: scripts de instalacion y operacion.

## Inicio rapido

1. Versiona este vault en un repo privado.
2. Ejecuta `CursorMemory-Install.cmd`.
3. Reinicia Cursor.
