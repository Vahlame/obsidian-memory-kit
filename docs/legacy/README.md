# Legacy artifacts

- **`PROMPT_ULTRA_COMPLETO_v1.md`** — v1 operational prompt (Cursor + Windows-oriented). Prefer v2 `AGENTS.md` + initializer for new installs.
- **`windows-v1/`** — deprecated PowerShell watchdog / autosync and Task Scheduler XML exports from v1 (see README there).

These paths are retained for reproducibility and rollback; they are **not** maintained as first-class install paths.
