# Useful Prompts

## Bootstrap memory in a new chat

`Usa obsidian-memory. Lee MEMORY.md y PROJECTS/<proyecto>.md. Dame un resumen de contexto y riesgos antes de empezar.`

## Save checkpoint every 3-5 messages

`Haz checkpoint en PROJECTS/<proyecto>.md si hubo avance real. No escribas ruido ni duplicados.`

## Close task and persist decisions

`Cierra esta tarea guardando: (1) resumen en SESSION_LOG.md y (2) aprendizajes durables en MEMORY.md.`

## Diagnose memory availability

`Valida si obsidian-memory esta disponible. Si falla, explica si es problema de MCP config, health endpoint o scheduler.`
